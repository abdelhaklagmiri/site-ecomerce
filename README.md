# VALON Hair Care — Boutique Shopify

Thème premium luxe pour le marché marocain. Pack Complet **649 MAD** — COD — WhatsApp.

## Structure du projet

```
valon-theme/          ← Thème Shopify (SEUL dossier à importer)
valon-theme-final.zip ← ZIP prêt pour Shopify Admin
docs/                 ← Guides dans valon-theme/docs/
```

## Démarrage en 3 étapes

1. **Importer** `valon-theme-final.zip` → Shopify Admin → Thèmes → Importer → Publier
2. **Suivre** `valon-theme/docs/SHOPIFY-ADMIN-CHECKLIST.md` (étape par étape)
3. **Coller** les textes depuis `valon-theme/docs/TEXTES-COPIER-COLLER.md`
