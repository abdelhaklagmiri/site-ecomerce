## 1. Brand Voice

| Attribut         | Détail                                                                                  |
| ---------------- | --------------------------------------------------------------------------------------- |
| Ton              | Confiant, sensoriel, intemporel. Pas de jargon marketing.                              |
| Vocabulaire      | Précis. *« Fibre capillaire », « bulbe », « cuir chevelu », « kératine »*.              |
| Émotion          | Aspiration + soulagement. *« Vos cheveux méritent ce qu'il y a de mieux. »*             |
| Origine          | Toujours rappeler **Istanbul / Türkiye / formule turque**. C'est le pilier différenciant. |
| Mention darija   | Permise dans les témoignages clients pour l'authenticité (ex : *« produit nadi »*, *« wallah… »*) |
| À éviter         | Promesses non vérifiables (« 100% efficace »), expressions dropshipping (« en rupture imminente », fausses minuteries excessives) |
